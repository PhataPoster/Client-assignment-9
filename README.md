# DocAppoint

DocAppoint is a modern doctor appointment booking platform built with Next.js. It helps users browse doctors, manage bookings, and handle appointments through a smooth, responsive client experience.

Live site: https://client-doc-appointment.vercel.app

## Features

- Browse doctors and explore appointment details in a clean, responsive UI.
- Book, update, and delete appointments from a simple client-side workflow.
- Sign up and sign in with authentication support for protected actions.
- Receive toast notifications for successful booking actions and updates.
- Use a responsive banner, navbar, and modal flow that works across screen sizes.
- Keep booking data normalized so the interface stays stable even when the API returns different shapes.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the app locally.
